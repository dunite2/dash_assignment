Welcome to the Flask Image Converter & QR Generator!

This application provides two main functionalities:

1. Image Format Converter

Route: /convert (POST method)

Description: Convert images from one format to another using the Pillow package.

How to use:

  1. Select an image file (PNG, JPG, JPEG, GIF, BMP, TIFF, WEBP)
  2. Choose the desired output format
  3. Click "Convert Image" to download the converted file

2. QR Code Generator

Route: /generate_qr (GET and POST methods)

Description: Generate QR codes from text using the pyqrcode package.

Features:

  • Convert any text into a QR code
  • Adjustable error correction levels (L, M, Q, H)
  • Customizable scale/size
  • Downloads as PNG image

Error Correction Levels:

  • L: Low (~7% correction)
  • M: Medium (~15% correction) - Default
  • Q: Quartile (~25% correction)
  • H: High (~30% correction)

API Usage Examples

You can also use these routes programmatically:

Convert Image (POST):

curl -X POST -F "image=@your_image.jpg" -F "output_format=PNG" http://localhost:5000/convert

Generate QR Code (POST):

curl -X POST -F "text_data=Hello World" -F "error_correction=M" -F "scale=8" http://localhost:5000/generate_qr
